# 🚀 Cosmic Carnage

> A fast-paced endless runner built in Unity where every split-second decision matters.

[🎮 Play the Game (Itch.io Web Build)](https://manit1863.itch.io/cosmic-carnage) • [🎥 Watch the Trailer](https://www.youtube.com/watch?v=QfduDJBMT9s)

## 📖 About the Project

**Cosmic Carnage** is a 3D endless runner that combines obstacle avoidance with shooting mechanics. Players must dodge incoming hazards while identifying and eliminating hostile enemies to earn points.

Not every target should be shot, however. Friendly animals occasionally appear and shooting them results in score penalties, forcing players to make quick decisions under pressure.

As the game progresses, movement speed increases, creating a more challenging and rewarding experience.

---

## Features

*  Endless runner gameplay with increasing difficulty
*  Real-time shooting mechanics
*  Multiple enemy types with unique score values
*  Friendly animals that penalise incorrect shots
*  High-score focused gameplay loop
*  Dynamic obstacle avoidance
*  Audio and visual feedback systems
*  Complete UI system including menus and game over screens

## Development Journey

This project was created as part of a university game development assessment.

The initial concept was inspired by Unity's *Create with Code* learning pathway. I expanded on those concepts by combining endless-runner mechanics with target identification and shooting systems to create a more engaging gameplay experience.

The project allowed me to strengthen my skills in Unity development, C# programming, gameplay system design, debugging, and project delivery.
